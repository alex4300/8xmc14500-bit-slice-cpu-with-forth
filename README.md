# 8xMC14500-Bit-Slice-CPU-with-Forth
An 8-bit CPU built from eight 1-bit ALU slices, inspired by the Motorola MC14500 Industrial Control Unit. Microcode-driven. Includes a complete Forth interpreter/compiler that runs on the CPU, runs on real hardware (Tang Primer 20K FPGA), and boots a block-based on-chip filesystem with a hex editor.
